# eslint-config-colinrcummings

ESLint rules for my personal projects.

## Usage

Install packages:

```
npm install --save-dev eslint eslint-config-colinrcummings
```

Add "extends": "colinrcummings" to your `.eslintrc`:

```javascript
{
  "extends": "colinrcummings",
  "rules": {
    // your overrides
  }
}
```
